Tabular Recipe Notation

A visual format for recipes: steps flow left-to-right, ingredients flow top-to-bottom

How to Use

  • Edit the JSON in the left panel to see the table update in real-time
  • Use colspan to span multiple columns (for preparation steps)
  • Use rowspan to span multiple rows (for actions that apply to multiple ingredients)
  • Use class: "vertical" to rotate text 90 degrees
  • Use class: "righthide" to hide the right border for visual continuity
  • Use align: "center" to center text in a cell
